# Quickstart: Tag Variant Management

## Prerequisites

- API and web apps running.
- A story with at least one scene.

## Manual Test Steps

1. Open a scene and click Manage in the Tags section.
2. Convert a non-variant tag to a variant via the branch icon.
3. Verify the row switches to variant mode and can be expanded.
4. Expand the tag and add a variant.
5. Select the variant for the scene and confirm the row shows a variant badge when collapsed.
6. Attempt to delete a variant in use and confirm it is blocked with an error message.
7. Delete a variant that is not in use and confirm it disappears.
8. Collapse and re-open the modal; verify the selected variant remains.
